Игра где нужно находить ключи и решать загадки, чтобы открыть сундук с сокровищами.

Установка:
make install

Запуск:
make project

Команды:
go north/south/east/west - двигаться
look - осмотреться
take предмет - взять предмет
use предмет - использовать
inventory - инвентарь
solve - решить загадку
help - помощь
quit - выход

Ответы на загадки: 
Молчание
Резонанс
Остальные легкие
